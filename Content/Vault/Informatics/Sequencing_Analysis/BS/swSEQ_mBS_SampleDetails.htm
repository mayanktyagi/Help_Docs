Sample Details Page

The Sample Details page allows you to see the results for a particular sample in an app session. The panes shown depend on the app run; for a description, see Sample Details Page Components

You can also launch apps on your sample, and open the Sample Overview page.

In addition, there is a Side Navigation ribbon, which provides easy navigation in the Project Details area. It contains links to the App Results, Sample Details Page, and associated files.
