Share

Data in BaseSpace can be shared with collaborators in a couple of different ways. You can either share data at a run or project level, via an email invitation or through a hyperlink. With the email invitation option, only the accounts with the specified email can view shared data. Sharing via a hyperlink option allows anyone with access to the hyperlink to be able to view the shared data, as long as the hyperlink is still active.

Sharing is for read-only access. If you want a collaborator to have write acces, see Transfer Ownership.

 

Note

Runs and projects have separate permissions. If you share a run you do not share the projects associated with the samples within that run.
