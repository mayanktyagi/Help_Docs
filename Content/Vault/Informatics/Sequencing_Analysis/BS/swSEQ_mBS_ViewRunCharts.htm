View Run Charts

What is it

The Charts page displays charts with run metrics.

When to use it

Use this when you want to view charts such as Flow Cell, Data By Cycle, Data By Lane, Qscore Distribution, and Qscore Heatmap. For more information, see Charts.

Why to use it

Use this if you want access to these various charts.

How to use it

    1 Click the Runs icon.
    2 Click the desired run.
    3 There are two methods to go to the Charts page:
      • From the Run Overview page click the Charts link.
      • From the Run Overview page click the Charts icon from the sidebar navigation menu.
