Project Overview Page

The Project Overview page provides 4 panes:

    • The Project Details pane gives a summary of the project with links to do the following:
      • Apps: Allows you to run custom apps on your sample. Clicking on the app name leads to a page with more information about launching that app, including access permissions.
      Note that running custom apps may incur a charge.
      • Edit project details: Allows you to edit the name and description of the project
      • Share: Allows you to manage sharing a project with a particular collaborator
      • Get link: Allows you to forward the sharing link to any number of collaborators
    • The Project Summary pane gives you the number of samples, App Sessions, and collaborators. For more information, see Project Samples List, App Sessions List, or Collaborator List.
    • The Samples pane gives a list of all the samples in the project. Clicking on the header will take you to the Sample List (Project Samples List), clicking on a sample links to the page for that sample (see Sample Overview Page).
    • The App Sessions pane gives a list of all the App Sessions in the project. Clicking on the header will take you to the App Sessions List (App Sessions List), clicking on a sample links to the app results for that sample (see App Results Page).

    In addition, there is a Side Navigation ribbon, which provides easy navigation in the Project Details area. It contains links to the overview, App Sessions, samples, and collaborator pages.
