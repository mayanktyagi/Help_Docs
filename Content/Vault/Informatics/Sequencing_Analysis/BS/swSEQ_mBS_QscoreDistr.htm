QScore Distribution

What is it?

The QScore Distribution Plot displays plots that allow you to view the number of reads by quality score. The quality score is cumulative for current cycle and previous cycles, and only reads that pass the quality filter are included.

When to use it.

Use it to judge the QScore distribution for a run.

When not to use it.

Do not use the QScore Distribution Plot to look at secondary or tertiary analysis metrics.

How to use it

The QScore Distribution plot has the following features:

    • You can select the displayed read, and cycle through the dropdown lists.
    • The symbol in the top right hand corner toggles the plot between pane view and full screen view.

Note that the QScore is based on the Phred scale. The following list displays Qscores and the corresponding chance that the base call is wrong:

    • Q10: 10% chance of wrong base call
    • Q20: 1% chance of wrong base call
    • Q30: 0.1% chance of wrong base call
    • Q40: 0.01% chance of wrong base call

You can slide the threshold (set at >=Q30 by default) to examine the proportion of bases at or above any particular Q-score, note that when using Q-score binning this plot will reflect the subset of Q-scores used. You can expand a chart by clicking on the expand button.
