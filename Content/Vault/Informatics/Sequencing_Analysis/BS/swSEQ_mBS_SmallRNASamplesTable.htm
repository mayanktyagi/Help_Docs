Small RNA Samples Table

Column

Description

Sample Name

The sample name from the sample sheet.

Cluster Raw

The number of raw clusters detected for the sample.

Cluster PF

The number of clusters passing filter for the sample.

Cluster Align Contam

The number of clusters that match records in the Contaminants database.

Cluster Align miRNA

The number of clusters that exactly match records in the Mature miRNA database.

Cluster Align RNA

The number of clusters that match records in the RNA database.

Cluster Align Genome

The number of clusters that match records in the genomic database.

Cluster Unaligned

The number of clusters that did not align against any reference database.
