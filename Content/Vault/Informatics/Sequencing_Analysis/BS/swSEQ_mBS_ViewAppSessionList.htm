View the App Sessions List

What is it

The App Sessions List contains a list of app sessions in a project.

When to use it

Use this when you want to navigate to details regarding a specific app session.

Why to use it

Use this when you want to see more details regarding an app session.

How to use it

    1 Click the Projects icon.
    2 Click the desired project.
    3 There are three methods to go to the App Sessions List:
      • From the Project Overview page click the App Sessions link.
      • From the Project Overview page click the App Sessions icon from the sidebar navigation menu.
      • From the Project Overview page click App Sessions icon.

You can now click on an App Session to see the results; for more information, see App Results Page.
