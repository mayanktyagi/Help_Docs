Set Up a New Project

What is it

A methodto set up a new project.

When to use it

    • When you want to analyze a sample in the context of two different projects
    • When you want to transfer ownership of samples to a collaborator, but still keep a copy yourself
    • When you want to split a project into multiple projects

How to use it

    1 Click the Projects icon.
    2 Click New Project button in the top left corner.
    3 Enter a new name and description.
    4 Click the Create button.

To copy samples into the new project, seeCopy Samples.
