Download File Package from a Run

What is it

BaseSpace allows you to download data either as a package or individually. This topic describes how to download packages; for individual FASTQ files, see Download FASTQ Files with the File Browser on page 1.

The packages available depend on your workflow; packages that are grayed out are not available for download. There are four types of data packages:

    • Variant Data, containing vcf files with variant calls.
    • Aligned Data, containing BAM files with aligned reads.
    • Unaligned Data, containing FASTQ files with unaligned reads.
    • SAV Data, containing file describing the set up of the run and interop files.

For more information about file types, see BaseSpace Files.

When to use it

    • Use this when you want to download a packaged(zipped) file for Variant, Aligned, Unaligned or SAV data.
    • Don’t use if you only want individual files.

Why to use it

If you want to download for Variant, Aligned, Unaligned or SAV data in a neatly packaged file versus downloading the files one-by-one.

How to use it

    1 Click the Runs icon.
    2 Click the desired run.
    3 Click the Download button.
    4 Select the desired data option.
